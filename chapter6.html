Introdução

Esse capítulo visa estabelecer conceitos importantes que envolvem a linguagem C, compiladores e programação no geral.

Sample C Program

/* hello.c */
#include <stdio.h>
int main() {
    printf("Hello, World!\\n");
    return 0;
}

Compiling with GCC

$ gcc -o hello hello.c
$ ./hello
Hello, World!
